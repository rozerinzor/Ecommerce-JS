# 🛒 E-Commerce Web Project

Bu proje, **HTML**, **SASS** ve **JavaScript** kullanılarak geliştirilmiş basit bir e-ticaret arayüzüdür. Kullanıcı dostu bir tasarım ve temel e-ticaret işlevlerini öğrenmek/pratik yapmak amacıyla hazırlanmıştır.
